心山月
Created 2019-11-25| Updated 2023-11-06|Story
Post View:

我知道那不是山。

灯火阑珊

  • 你问我见到你怎么那么开心，我想了又想，可能是因为心想事成吧

何处为家

  • 未来，要养一只猫或者一只狗，总之你喜欢的就好，取名叫小鬼
  • 生一双儿女，叫小妖小怪
  • 别看了，你就是那个大魔头

时间

  • 读书时，我们每天相伴了十六个小时
  • 你总担心很快就要分离
  • 我只好说，会有很多很多，陪你二十四小时的时间

求婚

  • 我想你先拒绝我九十九次
  • 然后再答应我
  • 这样你就能感受这快乐一百次

最伤心的事

  • 你看不到这一句

愧疚

  • 我经常想起你
  • 却想不起任何你喜欢的东西
  • 我才明白愧疚源于何处

一池春水

  • 今天路过一座桥，看桥下水面波纹阵阵
  • 想起从前老师在讲台上，又拍又呼地说，吹皱一池春水，皱字极妙
  • 要我说，也没有别的字可用了

一生

  • 有人说，我娶你，比我爱你更令人感动
  • 可一想到你，我就觉得结婚不过是我们共同面对的一件小事
  • 我要你的一生，你可愿意？

想了太多

  • 我想，菩萨能算八十一难，那他一定能算到，我会想什么
  • 连同我在想菩萨算我想什么这件事本身
  • 那我应该在什么时候停止呢，幸好，对于菩萨而言，在任何时刻停止想象，都是命中注定
  • 可惜你比菩萨还要难对付，猜也是错，不猜也是错

忽远忽近

  • 说来也是好笑
  • 你喜欢的人在天的另一边
  • 竟让我体会到什么是伸出手却又收回
